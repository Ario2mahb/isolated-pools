### Prerequisites

* NodeJS - 12.x
* Solc - v0.8.13 (https://github.com/ethereum/solidity/releases/tag/v0.8.13)

### Installing

```
yarn install --lock-file
```

### Run Tests

```
yarn test
yarn hardhat:test
```
